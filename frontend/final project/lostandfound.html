FEATURES

Sign Up or Be a guest

Guest mode is for users only want to search lost items in our website. Personal account mode is designed for users to search and claim lost items, upload items they found. Most importantly, you can link a unique serial code to your account as an ID.

Upload Found Items

Provide all required information about the items you found and upload it to our website. We will match your submission to a search request when someone searchs similar lost item information. Help other people is always a good thing!

Serial Code

You can generate and claim a unique serial code from our website for your account. Put it on all your belongings so no other people can claim your stuff if you lose them. Only the account linked with the correct serial code can claim corresponding items.

Automatch Lost Items

Fill in required information about the item you lost in the search page and submit it. Our system will automatically return all possible matches. Browse through the refined results to determine which one is yours. Simple and fast!
